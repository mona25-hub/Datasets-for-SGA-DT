# Datasets-for-SGA-DT
All the data used in the manuscript are available in this repository.

The authors confirm that all data used in the experiments are fully available in open
data repositories (KEEL, UCI, and Kaggle) and the links to the datasets are as below:

[1] Breast Cancer Dataset, KEEL Dataset Repository, 2011.
https://sci2s.ugr.es/keel/dataset.php?cod=97.
[2] KEEL Dataset Repository, Mammographic Dataset, 2011.
https://sci2s.ugr.es/keel/dataset.php?cod=86.
[3] UCI Machine Learning Repository. Hepatitis data set, 1988.
https://archive.ics.uci.edu/dataset/46/hepatitis
[4] Fede Soriano, Air quality data set, Kaggle repository, 2021.
https://www.kaggle.com/datasets/fedesoriano/air-quality-data-set
[5] Tawfik Elmetwally. (Adult dataset), 2023.
https://sci2s.ugr.es/keel/dataset.php?cod=192
[6] Dua Dheeru and Efi Karra Taniskidou. KDD Cup data UCI machine learning repository, 2017.
https://archive.ics.uci.edu/dataset/129/kdd+cup+1998+data
